
# Kreftdashboard Norge - Backend

## Installation
```bash
pip install -r requirements.txt
```

## Run locally
```bash
uvicorn main:app --reload
```

## API endpoints
- `/kreftformer`
- `/regioner`
- `/data?kreftform=Bryst&region=Helse Sør-Øst RHF&kjønn=Kvinner`
- `/statistikk`
